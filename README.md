# 🍔 Food Delivery App

A full-stack **Food Delivery** application built with the **MERN Stack** that allows users to browse restaurants, explore menus, add items to their cart, place orders, and manage restaurants through a modern and responsive interface.

---

## 🚀 Features

### 👤 Authentication

* User registration and login
* JWT-based authentication
* Password hashing with bcrypt
* Protected routes
* Role-based authorization (Customer, Restaurant Owner, Delivery Boy)

### 🍽️ Restaurant Management

* Create and manage restaurants
* Update restaurant details
* Upload restaurant images
* Categorize restaurants
* View restaurant listings

### 🍕 Menu Management

* Add food items
* Edit and delete menu items
* Upload food images
* Set pricing and availability

### 🛒 Cart

* Add items to cart
* Update item quantity
* Remove items
* Calculate total price

### 📦 Orders

* Place orders
* View order history
* Track order status
* Restaurant owner order management

### 📱 User Experience

* Responsive design
* Modern UI
* Fast API responses
* Form validation
* Error handling
